# GlitchLab Pro - AI Camera with Real-time Effects

An interactive camera application with real-time visual glitch effects and filters.

## Features

- 📷 Real-time camera effects
- 🎨 8+ Visual effects (Noise, RGB Shift, Scanlines, Pixelation, Wave Distortion, Barrel Distortion, Chromatic Aberration, Glitch)
- 🎬 6 Pre-made presets (Cyberpunk, VHS, Matrix, Glitch, Retro, Dream)
- 💾 Save custom presets to browser storage
- 🎥 Record videos with effects
- 📊 Real-time FPS counter
- 🔧 Adjustable parameters for all effects

## How to Use

1. Open `GlitchLab_Pro_Enhanced.html` in your web browser
2. Click "カメラ開始" (Start Camera)
3. Allow camera access when prompted
4. Adjust effects using the sliders on the left panel
5. Try different presets from the dropdown
6. Click "保存" (Save) to download screenshots
7. Click "録画" (Record) to record videos

## Effects

- **ノイズ (Noise)**: Random color noise overlay
- **RGBシフト (RGB Shift)**: Color channel displacement
- **走査線 (Scanlines)**: TV-like horizontal lines
- **モザイク (Pixelation)**: Block/pixelation effect
- **波打ち歪み (Wave Distortion)**: Wavy lines animation
- **樽歪み (Barrel Distortion)**: Curved lens effect
- **色収差 (Chromatic Aberration)**: Color fringing effect
- **グリッチ量 (Glitch)**: Random horizontal displacements
- **ポスター化 (Posterization)**: Color reduction effect
- **彩度 (Saturation)**: Color intensity control
- **明るさ (Brightness)**: Image brightness adjustment
